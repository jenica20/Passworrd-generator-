# Passworrd-generator-
A simple and customizable Password Generator built using Python.
This project allows users to choose what types of characters they want in their password — lowercase letters, uppercase letters, numbers, and symbols — and then generates a strong random password based on their choices.

This is a great beginner project to understand Python basics such as loops, conditionals, user input, and working with modules like random and string.
Choose the type of characters to include:

✔ Lowercase letters

✔ Uppercase letters

✔ Digits (0–9)

✔ Symbols (!@#$%...)

Set custom password length

Generates a unique random password every time

Beginner-friendly and easy to extend
How to Run

Install Python (if not already installed).

Save the code in a file named password_generator.py

Open terminal / command prompt in the same folder

Run:

python password_generator.py


Follow the on-screen instructions and enter:

y/n for each character type

password length
